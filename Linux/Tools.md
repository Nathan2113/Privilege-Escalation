**Linux Smart Enumeration**
[GitHub](https://github.com/diego-treitos/linux-smart-enumeration)

running with default gives the most important information
- will ask if you know the current user's password
- `-i` to not prompt password
![](assets/Pasted%20image%2020260531142132.png)
- if nothing from first run was fruitful, run level 1 > then level 2



**LinEnum**
[GitHub](https://github.com/rebootuser/LinEnum)

cool features
- able to copy interesting files for export
- search for certain keywords

`./LinEnum.sh -k password /e export -t`
